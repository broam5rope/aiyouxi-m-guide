# aiyouxi-m-guide

## 项目简介

`aiyouxi-m-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅用于存放和展示各类 HTML 页面资源。  
所有页面均为静态 HTML 文件，可直接通过 GitHub Pages 或其他方式浏览。

## 目录说明

```
aiyouxi-m-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面，命名清晰便于识别。
- `assets/`：存放多个页面共享的样式、脚本或图片资源，减少重复。
- `index.html`：可选的主页，可作为仓库内容的导航或说明页面。

## 页面归档说明

本仓库中的 HTML 页面均以独立文件形式存在，每个页面包含完整的结构与功能。  
归档内容可能包括但不限于：

- 信息展示页面
- 工具类页面
- 实验性或学习用页面
- 其他静态 HTML 资源

页面之间无依赖关系，可单独访问或部署。  
如需添加新页面，请遵循以下原则：

- 页面文件放入 `pages/` 目录
- 避免使用外部资源链接（可放入 `assets/` 目录）
- 文件名使用小写字母、数字和连字符，便于 URL 访问

## 维护说明

## 使用方式

1. 克隆或下载本仓库
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件
3. 或通过 GitHub Pages 部署后在线访问（需在仓库设置中启用 Pages 功能）
